# Ab Initio Molecular Dynamics Tutorial

A hands-on guide to setting up, running, and analyzing AIMD simulations using VASP, ASE, and pymatgen.

## What you'll learn

- How to set up AIMD input files for VASP
- How to analyze trajectories: radial distribution functions, mean squared displacement, and diffusion coefficients
- How to extract thermodynamic properties from AIMD runs

## Prerequisites

- Familiarity with DFT and VASP
- Basic Python (NumPy, Matplotlib)
- Access to an HPC cluster for running VASP

Browse the tutorials in the sidebar, or open any notebook directly in Google Colab.
